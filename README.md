# Earnings Volatility Analysis

Event-driven options analytics for earnings and macro catalysts.

## Quick Start

```bash
# Install
.venv/bin/python -m pip install -r event_vol_analysis/requirements.txt

# Run (test mode)
.venv/bin/python -m event_vol_analysis.main --test-data --output reports/test_report.html
```

## Documentation

Start here: `docs/README.md`

## Two Functions

1. **Analyze** — Pre-event regime classification and strategy ranking
2. **Research** — Event history, replay, and outcomes analysis
